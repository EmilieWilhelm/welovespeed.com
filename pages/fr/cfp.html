Partagez une expérience !

Cette année, l’appel à propositions concerne toutes les conférences qui se tiendront à Sud-Web 2018.

Il y aura encore une forme de curation, mais tous les sujets passeront par ce formulaire ; ceci dans le but que l’équipe fasse le choix de la programmation sous forme de concertation.

Toutes les formes de conférences sont possibles :

  • présentation avec support(s) visuel(s) ;
  • lightning talk ;
  • conférence gesticulée ;
  • activité de groupe…

Nous ne souhaitons pas que vous vous sentiez enfermé·e·s dans l’élaboration d’une présentation "traditionnelle". Trouvons des formats qui ont du sens et qui vous sont naturels !

Vous avez une ou plusieurs idées ? N’hésitez pas, vous avez jusqu’au : 16 février 2018 à 23:59 (heure de Paris) pour les proposer.

Nous vous rappelons que nous privilégions :

  • le vécu, le retour d’expérience ;
  • le sens de votre démarche ;
  • ce que vous avez appris de cette expérience, l’enseignement que vous en avez tiré ;
  • le changement à appliquer dès lundi.

Vous avez peur de vous lancer ? N’hésitez pas à nous contacter, nous vous accompagnerons pour que vous soyez fin prêts le jour J !
