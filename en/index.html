Why is Sud Web more human?

Because there is more to it than just technical conferences: we prefer stories and feedback because we chose to be more interested in people than in the tools they use.

Because we try to forge connections between the web and the rest of the world: technicians, bakers, artists... All the feedbacks interest us!

Because we cultivate the human bond: we encourage informal conversations and make sure that they occur in pleasant, family-friendly and circulating spaces.

Because we create the conditions for a self-managed human adventure: we use the barcamp codes to organize a two-day immersive adventure in a convivial place.
